Raspberry Pi 5

A Raspberry Pi 5 é um pequeno computador de placa única, muito utilizado em projetos de programação, robótica, automação e aprendizagem.

Principais características físicas

  • Processador mais rápido
  • Portas USB
  • Saída HDMI
  • Ligação Ethernet
  • GPIO para projetos eletrónicos
  • Entrada para alimentação USB-C

Vantagens

Este dispositivo é bastante útil para estudantes e programadores, porque permite criar projetos reais com baixo custo.

Conclusão

A Raspberry Pi 5 é uma excelente ferramenta para aprender Python, redes, Linux, eletrónica e desenvolvimento web.
