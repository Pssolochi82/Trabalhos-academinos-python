EXERCÍCIO 7 – CLASSE VOO

Implementação de uma classe Voo com:
- Gestão de passageiros embarcados
- Controlo de capacidade máxima
- Embarque e desembarque com validações
- Cálculo de vagas disponíveis
- Cálculo da percentagem de ocupação
- Estado do voo (deserto, vagas disponíveis, lotado)

Ficheiros:
- ex7_classe_voo.py  -> implementação da classe
- test_voo.py       -> testes unitários com unittest

Como executar os testes do exercício 7:
python -m unittest test_voo.py
